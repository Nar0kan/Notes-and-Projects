Password Generator

PassGen is my old project that helped to generate pseudo-random passwords and saved them directly to the clipboard.

Brief tech stack: Python, Tkinter, JSON

Modules:

Secrets & Random
Pyperclip
CustomTkinter
Password Generator

My goal was to create an exe file and I did it like really fast. But then I was wanting to adapt my password generator with setting.

I used secrets lib for more secure pseudo-random generator, json for settings file and pyperclip to save passwords in clipboard. Also I played a little bit with styles using CTk.

PS: Option secured/insecured is meaning whether the secrets or random library is in use.
